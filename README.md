# Scoundrel (Python Edition)

A fan-made Python recreation of *Scoundrel*, the solo dungeon-crawling card game originally created by by Zach Gage and Kurt Bieg.

Built using pygame, this project features a complete graphical user interface, animated card movement, sound effects, highscores, menus, and plans for expanded gameplay systems.

The original print-and-play version of the game can be found here:  
[Scoundrel.pdf](http://www.stfj.net/art/2011/Scoundrel.pdf)

---

## Features

- Faithful recreation of the original gameplay
- Full graphical user interface
- Animated card interactions
- Sound effects and music
- Main menu and options screens
- Highscore tracking

---

## Planned Features

Planned additions and experimental ideas include:

- Character classes
- Roguelike progression mode
- Additional enemy and item mechanics
- Improved UI and visual polish
- Hover effects and deck preview
- Resolution scaling improvements
- Save system and run statistics

---

## Installation

### Requirements

- Python 3.x
- pygame

### Install dependencies

```bash
pip install pygame
```

### Run the game

```bash
python main.py
```
